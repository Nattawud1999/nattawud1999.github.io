# Nattawud | Portfolio

ยินดีต้อนรับสู่เว็บไซต์ Portfolio ของผม! เว็บไซต์นี้แสดงถึงโปรเจกต์และทักษะต่าง ๆ ที่ฉันพัฒนาในช่วงเวลาที่ผ่านมา ซึ่งเน้นการพัฒนาเว็บไซต์และแอปพลิเคชันที่มีความโต้ตอบและใช้งานง่าย โดยใช้เทคโนโลยีที่ทันสมัย เช่น **React.js** และ **Node.js**

## เกี่ยวกับฉัน

สวัสดีครับ! ฉันคือ **Nattawud** นักพัฒนาซอฟต์แวร์ที่หลงใหลในการสร้างเว็บไซต์และแอปพลิเคชันที่ใช้งานง่ายและมีประสิทธิภาพ ความเชี่ยวชาญของฉันรวมถึง:

- **การพัฒนา Frontend**: React.js, HTML5, CSS3, JavaScript
- **การพัฒนา Backend**: Node.js, Express, MongoDB
- **การออกแบบเว็บไซต์**: UX/UI Design, การออกแบบที่รองรับหลายอุปกรณ์ (Responsive Layouts)
- **การควบคุมเวอร์ชัน**: Git, GitHub

ฉันมุ่งมั่นที่จะพัฒนาทักษะและเรียนรู้เครื่องมือใหม่ ๆ ที่จะช่วยให้ฉันสามารถพัฒนาเว็บที่ดีขึ้นและตอบสนองต่อความต้องการของผู้ใช้งาน

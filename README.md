# แฟ้มลับ | Portfolio รูปแบบนักสืบ

เว็บพอร์ตโฟลิโอ (Portfolio) หน้าเดียวสไตล์ "แฟ้มคดีลับ" หรือ "ผลงาน" ที่มีความทันสมัยและใช้งานได้จริง
สร้างด้วย HTML, CSS และ JavaScript ล้วน 100% ไม่ต้องทำบิวด์ระบบด้วย Node.js 

## 📌 ฟีเจอร์หลัก
- **ธีมแฟ้มคดีลับ (Classified Dossier):** ระบบโทนดาร์กสุดเท่
- **ระบบ 2 ภาษา (Bilingual):** รองรับภาษาไทย (TH) และอังกฤษ (EN) พร้อมปุ่มกดมุมขวาบนเปลี่ยนภาษาอัตโนมัติ
- **แฟ้มผลงาน (Projects Section):** หน้ากระดานเปลี่ยนโฉมให้คุณใส่ "รูปภาพผลงาน" พร้อมซอร์สโค้ดและคำอธิบาย
- **ระบบหน้ามืด/สว่าง:** ปุ่มสลับโหมดกลางคืน (จอแฮกเกอร์) และโหมดสว่าง (แฟ้มกระดาษมะนิลา)
- **Responsive 100%:** รองรับทั้งหน้าจอมือถือ ไอแพด และเดสก์ท็อป

## 🚀 วิธีการเปิดใช้งาน
คุณไม่จำเป็นต้องมี Node.js, Vite หรือ React เพียงทำตามวิธีง่ายๆ ดังนี้:

**วิธีที่ 1 : เปิดไฟล์โดยตรง (ง่ายที่สุด)**
1. ดับเบิ้ลคลิกที่ไฟล์ `index.html`
2. หน้าเว็บจะเปิดขึ้นมาบนเบราว์เซอร์ของคุณทันที

**วิธีที่ 2 : ใช้ Live Server (แนะนำหากต้องการแก้ไขสลับดูโค้ด)**
1. เปิดโฟลเดอร์ผ่านโปรแกรม **VS Code**
2. ติดตั้งส่วนเสริม **Live Server**
3. คลิกขวาที่ไฟล์ `index.html` แล้วเลือกเมนู **"Open with Live Server"**

## � วิธีแก้ไขประวัติของคุณและใส่รูปผลงาน

⚠️ **สำคัญเกี่ยวกับการแก้ไขข้อความภาษาไทย/อังกฤษ:**  
เนื่องจากเว็บนี้เป็นระบบ 2 ภาษา หากคุณต้องการแก้ไขข้อความ **ให้เข้าไปแก้ที่แอดเดรส `assets/js/main.js`** ด้านบนสุดของไฟล์จะมีโค้ด `const i18n = { en: {...}, th: {...} };` ให้แก้ไขข้อความภายในพจนานุกรมนี้ เพื่อให้ระบบกดคลิกปุ่มภาษาทำงานเสร็จสมบูรณ์

� **เปลี่ยนรูปภาพผลงาน (Projects Board):**
   - มองหาโซนโฟลเดอร์หลักแล้วสร้างโฟลเดอร์ `/assets/img/` เพื่อจัดเก็บไฟล์รูปของคุณ
   - เปิดหน้า `index.html` เลื่อนไปที่แท็ก `<section id="projects">`
   - มองหาแท็กบรรทัดแรกของงานต่างๆ `<img src="assets/img/project1.jpg"...>` และเปลี่ยน path ชี้ป้าย `src` ให้เป็นรูปที่คุณเตรียมไว้
   
👨‍💻 **เปลี่ยนรูปอวาตาร์ตัวโปรแกรมเมอร์ (Hero Section)**
   - ทับเปลี่ยนคลาสรูปเดิม `<svg class="avatar-svg"...>` ด้วยรูปภาพที่คุณต้องการ

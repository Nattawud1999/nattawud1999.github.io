# 🕵️‍♂️ Nattawud Portfolio - Classified Dossier Theme

ยินดีต้อนรับสู่โปรเจกต์ **Portfolio ส่วนตัวของคุณณัฐวุฒิ** (IT Support / Network Engineer)
เว็บไซต์นี้ออกแบบมาในธีม **"แฟ้มลับนักสืบ (Classified Dossier)"** ที่มีความแปลกใหม่ โดดเด่น และไม่เหมือนใคร เพื่อใช้สำหรับพรีเซนต์ตัวตนและผลงานสำหรับการทำงานด้านสายไอทีโดยเฉพาะ

### 🌟 ฟีเจอร์ที่น่าสนใจ (Features)

เว็บไซต์นี้ถูกสร้างโดยใช้ HTML, CSS และ JavaScript ล้วนๆ (Static Website) โดยไม่มีการพึ่งพา Framework หรือ Library ภายนอก ทำให้เว็บไซต์ทำงานได้ **รวดเร็วและลื่นไหล** พร้อมด้วยลูกเล่น (Gimmick) มากมายดังนี้:

1. **💻 ระบบ Terminal Boot Sequence**
   - หน้าจอจำลองการเชื่อมต่อเข้าระบบ (Fake Login Terminal) ก่อนเข้าสู่เว็บไซต์
   - มีข้อความวิ่งพร้อมเสียงคีย์บอร์ดเสมือนการขอเข้าถึงข้อมูลระดับสูง

2. **🌐 ระบบสลับภาษา (TH / EN)**
   - สลับเนื้อหาทั้งเว็บได้ทันทีระหว่างภาษาไทยและภาษาอังกฤษ
   - ข้อมูลและเนื้อหาทั้งหมดถูกเขียนอยู่ในรูปแบบพจนานุกรมในไฟล์ `main.js` ทำให้แก้ไข/อัปเดตได้ง่ายดาย

3. **🌓 ระบบสลับภาพลักษณ์ (Dark / Light Mode)**
   - **Dark Mode:** พื้นหลังกระดานหลักฐานสีเข้ม พร้อมแฟ้มคดีปกสีเทา
   - **Light Mode:** พื้นหลังกระดานไม้ก๊อก พร้อมแฟ้มปกสีน้ำตาล (Manila Folder)
   - ระบบจะจดจำค่าที่ผู้ใช้เลือกไว้ล่าสุดใน Local Storage สะดวกต่อการกลับเข้ามาดูซ้ำ

4. **🔊 ระบบเสียงประกอบ (Web Audio API SFX)**
   - เสียงตรายาง "ปึ้ง!" ตอนกดเข้าถึงแฟ้ม
   - เสียงคีย์บอร์ด และเสียงปุ่มกดต่างๆ เพื่อเพิ่มมิติความสมจริงให้กับประสบการณ์ผู้ใช้

5. **🎯 เป้าเล็งนักสืบ (Custom Native Cursor)**
   - เมาส์แบบพิเศษไม่เหมือนใคร ไม่มีอาการดีเลย์
   - เมาส์ปกติเป็นรูป "เรดาร์" และเมื่อชี้ที่ปุ่มหรือลิงก์จะเปลี่ยนเป็นรูป "เป้าเล็งสีแดง"

6. **📁 แอนิเมชันเลือกและดาวน์โหลดเรซูเม่ (Resume Download)**
   - มีระบบ Popup สำหรับเลือกภาษาของเรซูเม่ที่จะดาวน์โหลด (TH/EN)
   - แอนิเมชันตรายางคำว่า **"อนุมัติ (APPROVED)"** กระแทกหน้าจอก่อนทำการโหลดไฟล์ PDF เข้าเครื่อง

7. **🛜 แอนิเมชันแสงเลเซอร์ข้อมูลเครือข่าย (Network Data Beam)**
   - ภายในหน้าประวัติการทำงาน (Timeline) มีลูกเล่นเส้นแสงสีเขียววิ่งไหลลงมา สื่อถึงการเชื่อมต่อข้อมูลในสายงาน Network Engineer 

8. **💬 ระบบ Popup LINE เพื่อดู QR Code ติดต่อ**
   - เมื่อกดช่องทางติดต่อทาง LINE จะมีหน้าต่าง QR Code เด้งขึ้นมากลางจอ
   - ใช้งานง่าย สวยงาม ไม่ต้องกระโดดไปหน้าอื่น

### 📂 โครงสร้างของระบบไฟล์ (File Structure)
```
/
├── index.html                 # โครงสร้างหลักของหน้าเว็บ
├── README.md                  # เอกสารอธิบายการใช้งานต่างๆ
└── assets/                    # ที่เก็บไฟล์ที่จำเป็นทั้งหมด
    ├── css/
    │   └── style.css          # ไฟล์ปรับแต่งสไตล์ทั้งหมด
    ├── js/
    │   └── main.js            # ไฟล์ควบคุมเอฟเฟกต์ ลูกเล่น และพจนานุกรมแปลภาษา
    ├── img/                   # รูปภาพประกอบหน้าเว็บ และผลงานทั้งหมด
    └── resumeTH.pdf / resumeEN.pdf # ไฟล์เรซูเม่สำหรับใช้ดาวน์โหลด
```

### 🚀 วิธีการทดสอบและนำขึ้นออนไลน์
1. **ทดสอบในเครื่อง:** เพียงคุณ Double Click ที่ไฟล์ `index.html` ก็สามารถเปิดดูเว็บไซต์ผ่านเบราว์เซอร์ได้ทันที
2. **การอัปโหลดขึ้นเซิร์ฟเวอร์ (เช่น GitHub Pages):** สามารถโยนไฟล์ `index.html` และโฟลเดอร์ `assets/` รวมถึงไฟล์ต่างๆ ทั้งหมดนี้ขึ้นบนระบบโฮสติ้ง หรือ GitHub ของคุณเพื่อออนไลน์ได้ทันทีโดยไม่ต้องตั้งค่าฐานข้อมูลใดๆ
